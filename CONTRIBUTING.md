# Contributing to HNIVision

Thank you for your interest! HNIVision is in early development — the contributing
guide will be expanded as the package matures.

## For now

If you'd like to contribute or report an issue, please:

1. Open an [issue](https://github.com/LabMingzeChen/HNIVision/issues) to discuss before opening a PR
2. Fork the repo, create a feature branch from `main`
3. Submit a PR with a clear description of what changed and why

## Code of conduct

Be kind, be precise, be open. This is an academic project and we welcome
researchers at all levels.
